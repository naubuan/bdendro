BDendro documentation
=====================

:Version: |release|
:Release date: |today|

Bokeh helpers for visualizing a dendrogram.

Requirements
------------

* `Python <https://www.python.org/>`_ >= 3.6
* `Bokeh <https://bokeh.org/>`_ >= 1.1.0
* `NumPy <https://numpy.org/>`_ >= 1.10.1
* `SciPy <https://scipy.org/scipylib/>`_ >= 1.0.0

Installation
------------

.. code-block:: shell

   pip install git+https://github.com/naubuan/bdendro.git
